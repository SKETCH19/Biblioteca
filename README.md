## GitHub Pages (demo estatico)
Este proyecto incluye una version estatica para GitHub Pages en la carpeta `Biblioteca/docs`.

Pasos:
1. En GitHub, ve a Settings -> Pages.
2. Source: `Deploy from a branch`.
3. Branch: `main` y folder `/docs`.
4. Guarda y espera a que se publique.
